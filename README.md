# Daniel Deegan

**Cybersecurity Engineer · IAM Specialist**

Cyber Security Engineer at Octopus Energy, focused on protecting energy
infrastructure and customer data. I work across offensive and defensive
security — penetration testing, digital forensics, threat modelling, and
identity & access management — and build full-stack web platforms when the
problem calls for it.

---

### Focus Areas

- **Offensive Security** — penetration testing, vulnerability discovery & exploitation, red-team methodology
- **Digital Forensics & Malware Analysis** — reverse engineering, incident reconstruction, IOC development
- **Risk & Compliance** — ISO/IEC 27005, GDPR, PCI-DSS-aligned risk assessment
- **Identity & Access Management** — scalable, security-first IAM architecture
- **Full-Stack Engineering** — production web platforms with secure backends and payment integration
